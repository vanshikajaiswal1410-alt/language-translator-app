# 🌐 Hindi ↔ English Translator App

A simple and powerful Flutter-based Translator App that supports real-time text and voice translation between Hindi and English.

## 🚀 Features

- 🌐 Real-time Translation (Hindi ↔ English)
- 
- 🎤 Speech-to-Text (Voice Input)
- 
- 🔊 Text-to-Speech Output
- 
- 📋 Copy Translated Text
- 
- 🔄 Language Swap Feature
- 
- 📱 Clean and Responsive UI
